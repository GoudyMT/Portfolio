jammming

First React built application which allows the user to create playlist from Spotify after pulling their api key after signing in.

  • React
  • JavaScript
  • CSS
  • HTML

Detailed Overview

React app which allows the user to connect to their Spotify account, pull api data, and then allow them to search spotify for songs, artis, or albums and then display it in a search result. After search results have populated the user may add songs to a playlist, edit the playlist name, and once adding songs is complete, they can save the playlist to their personal spotify account.

Features and Functionality

  1. React interface for reduced website reloading.
  2. Spotify API integration
  3. Interactive website
  4. Adjustable viewing device supported.
  5. Search results updated based on playlist track added or not.

Outcome and Reflection

While completing this project it helped me better understand the React application framework, how API integration can populate information, and the ability to deploy virtual DOM with netlify.
